# Dever de casa — Verbo *aller*

> Objetivo: praticar a conjugação do verbo **aller**, identificar as letras mudas no final das conjugações, e transformar frases afirmativas em interrogativas usando a inversão sujeito/verbo (incluindo o caso especial do "t" eufônico para **il**, **elle** e **on**).

---

## ✅ Parte 0 — Conjugação do verbo *aller* (já feito)

As letras no final das conjugações abaixo **não se pronunciam**.

- Je vais
- Tu vas
- Il / Elle / On va
- Nous allons
- Vous allez
- Ils / Elles vont

## Lembretes rápidos

### Conjugação do verbo *aller* (ir)

| Pronome       | Conjugação |
| ------------- | ---------- |
| je            | vais       |
| tu            | vas        |
| il / elle / on | va        |
| nous          | allons     |
| vous          | allez      |
| ils / elles   | vont       |

### Inversões já treinadas

- Je vais → **Vais-je ?**
- Tu vas → **Vas-tu ?**
- Nous allons → **Allons-nous ?**
- Vous allez → **Allez-vous ?**
- Ils vont → **Vont-ils ?**
- Elles vont → **Vont-elles ?**

### Regra do "t" eufônico (il / elle / on)

Quando a inversão colocaria duas vogais em colisão (ex.: *va* + *il*), inserimos um **"t"** entre hífens para separar os sons:

- Il va → **Va-t-il ?**
- Elle va → **Va-t-elle ?**
- On va → **Va-t-on ?**

> O "t" não tem significado — ele existe apenas para evitar o encontro de duas vogais, algo que o francês evita sempre que possível.
